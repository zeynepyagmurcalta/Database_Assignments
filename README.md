# Database Assignments

This repository contains my SQL assignments for the Database Systems course.

## Files
- `createNewDatabase.sql` — Creates tables and shows JOIN examples.
- `school.sql` — Students/Courses/Enrollments database with INNER / LEFT / RIGHT JOIN queries.
- `LuckyRentACar.sql` — Car rental database assignment.
- `WMA_Assignment03.sql` — Additional SQL tasks.

## Usage
Open any `.sql` file in **SQL Server Management Studio** and run it.

## Tools
- SQL Server Management Studio (SSMS)
- T-SQL
